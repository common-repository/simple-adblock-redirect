AdBlockly redirect users whose uses AdBlocker when visiting your website.

== Description ==

When you're displaying ads on your website, some users may uses some kind of adblocker. The use of adblocker could minimize your website revenue through ads. This plugin allows website admin to redirect users whose using adblocker on your website.

This plugin has a very small impact on your website loading time and could effectively detect major adblocker.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/adblockly` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the AdBlockly screen to configure the plugin
4. Change desired landing page options.


== Frequently Asked Questions ==

= What ad services is supported =

This plugin uses AdSense blocking rule to detect adblocker. This means this plugin is built for AdSense. However, other ad services are possibly supported.
